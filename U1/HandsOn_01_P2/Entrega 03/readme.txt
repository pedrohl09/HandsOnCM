Este código simula a cobertura de uma rede celular utilizando o modelo de sombreamento correlacionado para avaliar o impacto da variação do parâmetro de correlação em diferentes regiões de cobertura. Ele calcula a potência recebida em cada ponto de medição considerando os efeitos de sombreamento aleatório e sombreamento correlacionado, além da perda de percurso com o modelo Okumura-Hata.

Para rodar o código, é necessário ter numpy e matplotlib instalados. A execução mostra as diferenças na potência recebida sob diferentes condições de sombreamento correlacionado, sendo útil para estudos de cobertura em redes celulares.

Com as bibliotecas instaladas, navegue até a pasta onde o arquivo "HandsOn_01_P2_3.py" está localizado utilizando o comando "cd /caminho/para/sua/pasta". Após acessar o diretório correto, execute o script com o comando "HandsOn_01_P2_3.py" (ou "HandsOn_01_P2_3.py" se estiver utilizando Python 3). Caso ocorra algum erro relacionado ao pip, utilize o comando "python -m ensurepip --upgrade" para atualizá-lo.

Se tudo estiver configurado corretamente, o código será executado sem problemas.
