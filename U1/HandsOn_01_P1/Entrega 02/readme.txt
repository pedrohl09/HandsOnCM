# HandsOn-01-v3

Este projeto contém um script Python chamado "HandsOn-01-v2.py" que requer o Python 3.x e as bibliotecas "numpy" e "matplotlib". Primeiro, verifique se o Python está instalado no seu sistema, digitando "python --version" ou "python3 --version" no terminal. Caso não tenha o Python instalado, faça a instalação antes de continuar. 

Para executar o script, será necessário instalar as bibliotecas mencionadas. No terminal, instale o "numpy" com o comando "pip install numpy" (ou "pip3 install numpy" para Python 3) e o "matplotlib" com "pip install matplotlib" (ou "pip3 install matplotlib"). 

Com as bibliotecas instaladas, navegue até a pasta onde o arquivo "HandsOn-01-v3.py" está localizado utilizando o comando "cd /caminho/para/sua/pasta". Após acessar o diretório correto, execute o script com o comando "python HandsOn-01-v3.py" (ou "python3 HandsOn-01-v3.py" se estiver utilizando Python 3). Caso ocorra algum erro relacionado ao pip, utilize o comando "python -m ensurepip --upgrade" para atualizá-lo.

Se tudo estiver configurado corretamente, o código será executado sem problemas.
