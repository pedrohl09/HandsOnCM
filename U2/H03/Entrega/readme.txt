# Nome do Projeto
Hands On 03
## Descrição

[Uma breve descrição do projeto]
Este script realiza simulações de sistemas de comunicação digital utilizando modulações BPSK e 16-QAM, incluindo a análise da relação sinal-ruído (Eb/N0) e a taxa de erro de bits (BER). A implementação utiliza conceitos de processamento de sinais e transformadas de Fourier para gerar, transmitir, e demodular sinais sob diferentes níveis de ruído com OFDM e sem OFDM. 


## Requisitos

* Python 3.10 ou superior
* IDE python (e.g. VS Code, pycharm, spyder, etc.)
* Numpy (https://numpy.org/)
* Matplotlib (https://matplotlib.org/)
* Scipy (https://scipy.org/)

Para executar o código, basta cumprir os requisitos acima e executar na IDE python de preferência ou executando o próprio arquivo no terminal. Para executá-lo no terminal, digite cd *path do arquivo handson.py* e aperte enter. É esperado que o código apresente os valores de variância do ruído e forneça gráficos referentes as constelações das modulações e gráficos comparativos de BER, Pe e Eb/N0.
