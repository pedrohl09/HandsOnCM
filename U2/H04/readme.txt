# Nome do Projeto
Modelagem da Taxa de Pico de Sistemas Modernos de Comunicação
## Descrição

[Uma breve descrição do projeto]
Este repositório fornece duas ferramentas para cálculo de throughput teórico em redes de comunicações móveis, implementadas em Python com interfaces gráficas criadas usando PyQt5. Os arquivos LTE.py e NR.py calculam o throughput para redes LTE (Long-Term Evolution) e 5G NR (New Radio), respectivamente.

GitHub: https://github.com/pedrohl09/HandsOnCM/tree/main/U2/H04


## Requisitos

* Python 3.10 ou superior
* IDE python (e.g. VS Code, pycharm, spyder, etc.)
* Python 3.7 ou superior
* PyQt5
* NumPy (para o LTE.py)

--------------------------------------------------------------------------------------------------------------
Para executar as ferramentas, é necessário ter Python 3.7 ou superior e instalar as dependências PyQt5 e NumPy. Basta executar os arquivos LTE.py ou NR.py no terminal, preencher os campos da interface gráfica com os parâmetros desejados e clicar no botão de cálculo para obter os resultados. Mensagens de erro serão exibidas em caso de valores inválidos ou combinações não suportadas.

Certifique-se de ajustar o caminho para o arquivo de tabela TBS (é exigido pelo LTE.py) no código antes da execução.
